# sprint7-final-project-mcfi
Se trata de un proyecto de práctica educativa en Python donde se trabaja desde la limpieza de datos, creación de columnas y análisis de datos con base a las preguntas requeridas.

**Objetivo**
  -Crear un reporte de la empresa 'ConnectaTel' en México y Colombia con el fin de entender como los clientes usaban realmente los servicios móviles de mensajes de texto y llamadas, buscando patrones de uso, comprender las necesidades del cliente en grupos segmentados y con esto mejorar la oferta mejorando la experiencia del usuario.

**Datasets utilizados**
  -plans.csv
  -users_latam.csv
  -usage.csv
  -user_profile.csv

**Etápas de Análisis**
  -Exploración de datos
  -Limpieza de datos
  -Agrupación de datos
  -Visualización de datos
  -Identificación de Outliers
  -Segmentación de Clientes
  - Visualización de Segmentación de Clientes
  - Análisis ejecutivo
